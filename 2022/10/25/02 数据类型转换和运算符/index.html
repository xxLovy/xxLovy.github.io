02 数据类型转换和运算符

数据类型转换和运算符

1. 输入

1.1 输入的语法

1.2 输入的特点

  1. 当程序执行到input时，等待用户输入，输入完成后再继续向下执行
  2. input接受用户输入后一般储存到变量，方便使用
  3. input会把接收到的任意用户输入的数据都当做字符串处理

2. 转换数据类型

2.1 转换数据类型的函数

可以看到在使用int(psw)时获取了一个int类型的数据，但psw的类型其实没有变化

不是所有字符串都可以转换成int或float

eval: 将str转换成它原本的类型

3. 运算符

  1. 分类
  • 算数运算符
  • 赋值运算符
  • 复合赋值运算符
  • 比较运算符
  • 逻辑运算符
  1. 优先级

()高于**高于* / // %高于+ -

3.1 算术运算符

运算符 描述
+ - 加，减
* / % // 乘，除，模，整除
** 指数

除法得到的结果一定是float类型这和C/C++，java等语言不一样。C/C++，java等语言的/相当于python的//

3.2 赋值运算符

运算符 描述
= 等于

3.2.1单个变量赋值

3.2.2多个变量赋值

3.3 复合赋值运算符

+= -= *= /= %= //= **= &= |= ^= >>= <<=

和其他语言基本相同

3.4 比较运算

表达式 描述
> 大于
< 小于
== 等于
>= 大于等于
<= 小于等于
‘!=’ 不等于

会返回一个bool类型的值

3.5 逻辑运算符

表达式 描述
and 大于
or 小于
not 等于

表达式之间的逻辑运算和正常的一样

数字之间的逻辑运算

and: 若一个数字为0则结果为0，否则结果为最后一个非0数字

or: 只有所有值为0结果才为0，否则结果为第一个非0数字
